csharp-sendpush
===============

Send push notifications natively for iOS, Android and Windows Phone 8, developed in C #.

Example
===============

var wp = new WindowsPhone("your_device_key", "yourMessage !!!", "yourViewInApp.xaml", TypeShipping.DeliverImmediate); 
wp.Send();
